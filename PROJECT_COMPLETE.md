## Getting Started

### Install Dependencies:
```bash
npm install
```

### Run Development Server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the website.

### Build for Production:
```bash
npm run build
npm start
```

## SEO Features Implemented:

1. **Meta Tags**: Comprehensive meta tags on every page
2. **Structured Data**: JSON-LD schema for organization and products
3. **Semantic HTML**: Proper heading hierarchy and semantic elements
4. **Image Optimization**: Next.js Image component with WebP/AVIF support
5. **Fast Loading**: Code splitting, lazy loading, optimized assets
6. **Mobile-Friendly**: Fully responsive design
7. **Open Graph & Twitter Cards**: Rich social media previews
8. **Sitemap Ready**: Clean URL structure for easy sitemap generation
9. **Keyword Optimization**: Strategic keyword placement throughout
10. **Alt Text**: All images have descriptive alt text

## Tech Stack:
- **Framework**: Next.js 14+ (App Router)
- **Language**: TypeScript
- **Styling**: CSS Modules + Global CSS
- **Fonts**: Google Fonts (Cinzel, Inter)
- **Images**: Unsplash (optimized via Next.js Image)
